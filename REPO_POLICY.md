# Repository Policy

## Visibility
- Visibility: Public
- Reason: FrameKit is intended for open ecosystem adoption.
- Promotion criteria to Public:
  - No secrets
  - Build/test instructions documented
  - CI active

## Commit Policy
- Keep commits small and focused by default.
- Large commits are only acceptable for atomic refactors that cannot be safely split.

## Tracking Policy
- Every implementation change must map to a GitHub project issue.
